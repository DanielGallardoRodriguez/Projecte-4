# T11: Introducció a Figma: nocions bàsiques de disseny d’interfícies

## Breu descripció
**Objectiu:** Conèixer Figma com a eina estàndard per al disseny d’interfícies i prototipat, i aprendre les funcions bàsiques per crear maquetes web i aplicacions.

---

## Introducció
Figma és una eina utilitzada per empreses tecnològiques per dissenyar:
- Interfícies web.
- Landing pages.
- Aplicacions mòbils.
- Fluxos de compra i experiències d’usuari.

**En aquesta tasca aprendrem:**
- Què és Figma i per què és l’eina estàndard en UX/UI.
- Com crear un projecte i organitzar pantalles (**frames**).
- Ús d’eines bàsiques: formes, text, colors, imatges i components.
- Creació d’estils (color, tipografia).
- Introducció al prototipat: enllaços, botons i navegació bàsica.
- Bones pràctiques de disseny per projectes web.

**Ampliació:**  
Si vols aprofundir, consulta aquest curs:
