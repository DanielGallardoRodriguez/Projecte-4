# T08: Auditoria de Qualitat i Estandardització de Servidors (tasca individual)

## Breu descripció
**Objectiu:** Consolidar la base dels servidors mitjançant una auditoria interna i aplicar el protocol d’estandardització (SOP) abans de desplegar serveis avançats.

---

## Introducció
Durant les darreres setmanes hem implementat serveis complexos (NFS, CUPS, LDAP). Però cap servei pot funcionar correctament si el sistema base és feble:
- Configuració de xarxa incorrecta.
- Permisos mal assignats.
- Paquets sense actualitzar.

A EverPia, la robustesa és clau. Per això, abans de desplegar nous serveis, hem de garantir que el servidor compleix el **Protocol de Desplegament Estàndard (SOP)**.

**Missió:**  
- Desplegar una màquina virtual Ubuntu Server des de zero.
- Aplicar el protocol SOP per assegurar configuració bàsica correcta.
- Preparar-se per la **certificació SOP-Specialist** mitjançant una prova pràctica.

---

## Fase d’entrenament
- Usar la OVA proporcionada.
- Realitzar les tasques del document **Projecte4-TascaSOP** (disponible per descarregar).
- No cal lliurar cap activitat, però és essencial comprendre bé els procediments.

---

## Certificació
- Examen pràctic individual per obtenir el nivell **SOP-Specialist**.
- Es permet portar **un full manuscrit** amb anotacions personals.

---

## Objectius específics
- Importació i configuració correcta d’Ubuntu Server a partir d’una OVA.
- Configuració de nom d’equip i xarxa.
- Actualització del sistema i instal·lació de paquets.
- Gestió de permisos sobre arxius i carpetes.
