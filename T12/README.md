# T12: Fonaments del Disseny Web Comercial: Landing Page + Procés de Checkout

## Breu descripció
**Objectiu:** Entendre l’estructura i els conceptes fonamentals de les pàgines web comercials, centrant-nos en dos pilars essencials: la **Landing Page** i el **Procés de Checkout**.

---

## Introducció
Abans de dissenyar la maqueta completa de l’e-commerce del client d’EverPia, cal comprendre:
- **Landing Page:** Pàgina d’entrada que capta l’atenció i converteix visitants en clients.
- **Procés de Checkout:** Flux que permet completar la compra de manera clara, segura i ràpida.

**Contingut de la classe magistral:**
- Estructura d’una landing page efectiva.
- Elements clau: titulars, CTA (Call to Action), beneficis, testimonis, garanties.
- Procés de checkout: passos, disseny intuïtiu, seguretat i UX.
- Bones pràctiques per maximitzar conversions.

**Exemples:**
- Landing page extensa: https://romualdfons.com/monetizacion-webs/
- Pantalles de checkout: exemples mostrats a la presentació.

---

## Objectius específics
- Comprendre com una landing page capta l’usuari.
- Analitzar com un checkout ben dissenyat facilita la conversió.
- Relacionar amb estratègies de disseny comercial web.
