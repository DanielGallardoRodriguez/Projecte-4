# T05: Accés Remot. Connexió via SSH (tasca individual)

## Breu descripció
**Objectiu:** Configurar el servei SSH per permetre la connexió remota segura entre equips Linux i Windows. Crear una guia interna que servirà com a base de coneixement per a futurs tècnics.

---

## Introducció
Els servidors dels nostres clients (i els nostres) rarament estan físicament accessibles. L’accés remot és essencial per:
- Administrar màquines de manera segura i eficient.
- Reduir temps de resposta davant incidències.

**Eina principal:**  
- **SSH (Secure Shell):** Estàndard de la indústria per administració remota xifrada, especialment en sistemes Linux.

**Missió:**  
Crear una **Prova de Concepte (PoC)** i documentar el procés per formar nous becaris. La guia ha d’incloure:
- Connexió SSH des de **Linux** (terminal nativa).
- Connexió SSH des de **Windows** (PowerShell o terminal moderna).

---

## Activitats a documentar
- Instal·lació i configuració del servei SSH.
- Generació i ús de claus pública/privada.
- Connexió entre sistemes Linux ↔ Windows.
- Proves d’accés i administració remota.
- Resolució d’incidències bàsiques.

---

## Objectius específics

- Documentar el procediment amb captures i explicacions clares.
