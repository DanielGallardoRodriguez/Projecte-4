# T06: Accés remot. Escriptori remot (RDP) (tasca individual)

## Breu descripció
**Objectiu:** Configurar el servei RDP per permetre connexions remotes segures amb entorn gràfic tant en equips Windows com Linux. Crear una guia tècnica clara per a futurs becaris.

---

## Introducció
En la tasca anterior vam treballar amb SSH (mode text). Ara ens centrem en **Escriptori Remot (RDP)**, imprescindible per:
- Administrar **Windows Server** amb interfície gràfica.
- Donar suport a usuaris finals que necessiten assistència visual (errors, aplicacions, etc.).

**Tecnologia principal:**
- **RDP (Remote Desktop Protocol):** Estàndard de Microsoft per connexions gràfiques.
- També disponible en entorns GNU/Linux (ex. Zorin amb Gnome).

**Missió:**  
Crear una **Prova de Concepte (PoC)** amb màquines virtuals i documentar:
- Connexió RDP entre sistemes Windows ↔ Linux.
- Configuració segura del servei.

## Objectius específics

- Documentar el procediment amb captures i explicacions clares.
